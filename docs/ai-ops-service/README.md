## 事業要旨

- **何を売るか**：顧客が作ったLLMアプリのプロキシ観測 + 月次運用代行
- **顧客**：30〜200名のSaaS／地場企業の情シス／士業
- **料金**：月額3万円〜30万円 + スポット50〜300万円
- **初年度目標**：年商1,860万円（MRR月155万円相当）
- **コア技術**：LiteLLM + Langfuse + Presidio（全てMIT）
- **Moat**：Claude Codeでも作れない領域（運用責任・コンプラ・IPレピュテーション・チャネル）
