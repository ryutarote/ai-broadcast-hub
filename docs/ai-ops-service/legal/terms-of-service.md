# Aegis サービス利用規約（ドラフト v0.1）

**ステータス**: Draft / **要弁護士レビュー**
**最終更新日**: 2026-05-14
**事業者名（プレースホルダ）**: 株式会社○○○○（以下「当社」）

---

## 第1条（適用）

1. 本規約は、当社が提供する「Aegis」と称するマネージドAIオペレーションサービス（以下「本サービス」）の利用に関する一切の関係に適用される。
2. 本サービスを利用しようとする者（以下「申込者」）は、本規約に同意のうえ申込みを行うものとする。
3. 当社が別途定める個別契約、SLA、データ処理委託契約（DPA）、プライバシーポリシー（以下総称して「個別規定」）が本規約と異なる場合、個別規定が優先する。

## 第2条（定義）

| 用語 | 定義 |
|---|---|
| 「契約者」 | 当社との間で本サービスの利用契約を締結した法人または個人事業主 |
| 「ユーザー」 | 契約者の従業員等で、契約者の管理下で本サービスを利用する者 |
| 「LLMプロバイダ」 | Anthropic, OpenAI, AWS Bedrock, Google Vertex AI 等、契約者が直接契約する大規模言語モデル提供事業者 |
| 「契約者データ」 | 契約者またはユーザーが本サービスに送信、または本サービスを通じてLLMプロバイダに送信するデータ全般 |
| 「契約者APIキー」 | 契約者がLLMプロバイダから発行を受けたAPI認証情報 |
| 「観測データ」 | 本サービスが記録するリクエスト・レスポンス・メトリクス・ログの総称 |

## 第3条（本サービスの内容）

1. 本サービスは、契約者が運用するAIアプリケーションとLLMプロバイダの間に位置する**プロキシ型観測プラットフォーム**および、それに付随する**運用代行サービス**を提供する。
2. 主な機能は以下のとおり：
   - LLMコールのプロキシ転送と観測
   - コスト・利用量・品質指標の可視化
   - 閾値アラート、フェイルオーバー、PII検知
   - 月次レポート、改善提案、緊急対応
3. 提供機能はプラン（Lite / Standard / Pro 等）によって異なり、最新内容は当社サイトに掲載する。

## 第4条（申込みと契約成立）

1. 申込者は当社所定の方法で申込み、当社の承諾通知をもって契約成立とする。
2. 当社は以下の場合、合理的判断により申込みを拒絶できる：
   - 反社会的勢力に該当または関係する場合
   - 過去に当社の契約を解除された者
   - 申込内容に虚偽がある場合
   - その他当社が不適切と判断した場合

## 第5条（料金と支払い）

1. 月額料金、従量料金、スポット料金は当社サイト掲載のとおり。
2. 月額料金は月初に当月分を請求、振込期日は請求日から30日以内とする。
3. 振込手数料は契約者負担とする。
4. 支払遅延の場合、年14.6%の遅延損害金を申し受ける。
5. **税抜表示**であり、別途消費税を申し受ける。インボイス制度（適格請求書発行事業者）登録番号は請求書に記載する。
6. 当社は90日前の通知により料金を改定できる。改定に同意しない契約者は通知到達から30日以内に解約できる。

## 第6条（契約期間と更新）

1. 契約期間は1ヶ月単位とし、解約申出なき場合は自動更新される。
2. 解約は退会希望月の前月末日までに当社に通知することで成立。
3. 月途中の解約による日割り返金は行わない（前払い分は当月末まで利用可）。
4. 年間契約プランを設けた場合は別途定める。

## 第7条（契約者の責任と禁止事項）

1. 契約者は以下を遵守する：
   - LLMプロバイダの利用規約・AUP（Acceptable Use Policy）を遵守すること
   - 契約者APIキーの管理と漏洩防止
   - 本サービスを通じて送受信するデータが法令・第三者の権利を侵害しないこと
   - 個人情報を含むデータを送信する場合は、個人情報保護法に基づく適切な取扱いを行うこと

2. 契約者は以下を行ってはならない：
   - 本サービスのリバースエンジニアリング、ソースコードの解析（OSS範囲を除く）
   - 本サービスのインフラに対する負荷試験・脆弱性診断（事前許諾あるものを除く）
   - 自動化された不正な大量リクエスト
   - 本サービスを通じた違法・有害なコンテンツの生成・配信
   - 他者のAPIキー・アカウント情報の不正取得・使用
   - 競合サービス構築を目的とした本サービスの利用

## 第8条（契約者APIキーの取扱い）

1. 当社は契約者APIキーを「契約者の指示に基づき、本サービスの提供のために必要な範囲」でのみ使用する。
2. 契約者APIキーはAWS Secrets Manager上で暗号化保管し、人間による直接参照は監査ログを取得する。
3. 契約者は当社にAPIキーを預託することについて、自らがLLMプロバイダとの契約上問題ないことを確認する責任を負う。
4. 当社はAPIキーをLLMプロバイダ以外の第三者に開示しない。

## 第9条（契約者データと知的財産）

1. 契約者データの一切の権利は契約者に帰属する。
2. 契約者は当社に対し、本サービス提供に必要な範囲で契約者データを利用する非独占的・無償・取消可能な使用許諾を与える。
3. 当社は契約者データを以下の目的に限って利用する：
   - 本サービス提供（プロキシ、観測、レポート）
   - 障害対応、セキュリティ調査
   - 法令に基づく開示要求への対応
4. **当社は契約者データをLLMモデルの学習に使用しない。また第三者に提供しない**（法令で要求される場合を除く）。
5. 観測データから生成された統計情報・改善提案については、契約者を識別できない匿名化処理を経たうえで、本サービスの改善・運用にのみ利用する。

## 第10条（個人情報・PII）

1. 当社のプライバシーポリシーおよびデータ処理委託契約（DPA）に従う。
2. 契約者は契約者データに個人情報を含めて送信する場合、事前にPIIマスキング機能の設定を行うこと、または契約者自身が個人情報保護法上の本人同意・通知義務を満たしていることを表明する。
3. 当社はPIIマスキング機能を提供するが、その完全性を保証しない。最終的な責任は契約者にある。

## 第11条（SLA）

1. 本サービスの可用性目標は別途定めるSLAに従う。
2. SLA未達の場合、当社は契約者に対して別途定める範囲のサービスクレジットを提供する。
3. 当該クレジット以外に当社は責任を負わない。

## 第12条（保証の限定）

1. 本サービスは「現状有姿」で提供される。当社は以下を保証しない：
   - 中断・エラーがないこと
   - 契約者の特定目的への適合性
   - LLMプロバイダの可用性・品質
   - LLM出力の正確性・有用性
   - PII検知の完全性
2. 本サービスはLLMプロバイダのAPI仕様変更・利用規約変更により機能変更される場合がある。
3. 当社の提案する「プロンプト改善」「モデル評価」は最善努力義務であり、結果を保証しない。

## 第13条（責任の制限）

1. 当社の契約者に対する責任は、原因の如何にかかわらず、**直近12ヶ月に契約者が当社に支払った料金の合計額**を上限とする。
2. 当社は、間接損害・特別損害・逸失利益（LLMプロバイダ側の障害による契約者の事業損失等を含む）について責任を負わない。
3. 故意または重過失による損害については、本条第1項の制限を適用しない。
4. 契約者が当社に通知すべきインシデントの通知を遅延した場合、当社の責任はその範囲で減免される。

## 第14条（情報セキュリティ）

1. 当社は本サービスの提供にあたり、合理的なセキュリティ対策を講じる（暗号化、アクセス制御、監査ログ、定期的な脆弱性診断等）。
2. 当社は外部委託（クラウド事業者、決済代行等）を行う場合、当該委託先に対しても適切な監督を行う。
3. セキュリティインシデント発生時、契約者へ速やかに通知する（個人情報漏洩の場合は個人情報保護委員会への報告も含む）。

## 第15条（業務委託先・サブプロセッサー）

1. 当社は本サービス提供のため、以下を含むサブプロセッサーを利用する：
   - Amazon Web Services Japan G.K.（インフラ）
   - SaaS事業者（請求、コミュニケーション）
2. 当社サイトに最新のサブプロセッサー一覧を公開する。
3. サブプロセッサーの追加・変更は契約者に30日前通知する。重大な追加に契約者が異議を申し立てた場合、双方協議のうえ対応を決定する。

## 第16条（一時停止と解除）

1. 当社は以下の場合、契約者への通知をもって本サービスを一時停止または契約解除できる：
   - 料金支払の遅延が30日を超えた場合
   - 第7条の禁止事項に違反した場合
   - LLMプロバイダ側で契約者の契約が解除された場合
   - 反社会的勢力との関係が判明した場合
   - 死亡・解散・破産手続開始等の場合

2. 当社は本サービスの全部または一部を、3ヶ月前の通知をもって終了できる。終了時、契約者にデータエクスポート機能を提供する。

## 第17条（届出事項の変更）

契約者は連絡先、商号、代表者等の届出事項に変更があった場合、速やかに当社に通知する。

## 第18条（譲渡禁止）

契約者は本契約上の地位・権利義務を、当社の事前書面承諾なく第三者に譲渡・移転・担保供与できない。当社が事業承継・合併等を行う場合、契約は承継先に承継される。

## 第19条（規約変更）

1. 当社は本規約を変更できる。変更内容と効力発生日を当社サイトに掲載することで効力を生じる。
2. **重大な変更（料金値上げ、サービス内容の大幅な変更、責任制限の強化等）は30日前**に契約者に通知する。
3. 契約者は変更後の規約に同意しない場合、効力発生日までに解約できる。

## 第20条（通知）

1. 当社から契約者への通知は、登録メールアドレス、管理画面、当社サイト掲載のいずれかによる。
2. 契約者から当社への通知は当社サイト掲載の問合せ窓口に行う。

## 第21条（分離可能性）

本規約の一部が無効と判断された場合も、残部の効力に影響を与えない。

## 第22条（準拠法・管轄）

1. 本規約は日本法に準拠する。
2. 本サービスに関する紛争は、**東京地方裁判所**を第一審の専属的合意管轄裁判所とする。

## 第23条（協議）

本規約に定めなき事項、または本規約の解釈に疑義が生じた場合、両当事者は信義誠実の原則に従い協議のうえ解決する。
