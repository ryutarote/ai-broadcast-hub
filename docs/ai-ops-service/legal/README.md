## 法務の論点サマリー

### 最重要

1. **顧客主体プロキシモデル**：Aegisは顧客のAPIキーで顧客の代わりにLLMを呼ぶプロキシ。再販モデルではない、と弁護士に確認をとる。
2. **OSSのAGPL対応**：Langfuse Coreは MIT 範囲で使う。Grafanaは社内利用に限定。
3. **責任制限**：12ヶ月料金を上限。個人情報漏洩時の例外を弁護士と協議。
4. **個情法27条委託**：DPAで委託先要件を明確化。
5. **LLMプロバイダ転送の整理**：「外国第三者提供」か「委託」かを弁護士と整理。

### 重要

6. **SOC2取得**：v1で着手、200〜500万円の予算確保。
7. **商標出願**：先願調査→出願。10〜30万円。
8. **サイバー保険**：年間20〜50万円で個人情報漏洩リスクをカバー。
