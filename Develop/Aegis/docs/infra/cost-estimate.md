### 2.2 コスト最適化（MVP段階で可能なもの）

- **Fargate Spot**: ワーカー系を Spot にし、$10〜30/月削減
- **CloudWatch Logs**: Retention 30日 + S3 へ移行で半額
- **NAT GW削減**: VPC Endpoint（S3/ECR/Secrets/KMS）拡充で 30〜50% カット
- **未使用リソース停止**: nights & weekends で staging を 停止 → $50/月削減

実質：**月額 10〜13万円に圧縮可能**

## 10. リスクと感度分析

### コスト増要因（影響大→小）

1. **顧客のトラフィック想定超過** → 自動スケールでコスト線形増（売上も増えるので問題小）
2. **NAT Gateway 通信量爆発** → VPC Endpoint で抑制
3. **CloudWatch Logs 過剰** → サンプリング・保持短縮
4. **データ保持期間長期化（Pro顧客）** → S3 IA/Glacier 自動移行
5. **LLM プロバイダの地理的拡張要求** → 海外リージョン展開でコスト2倍

### 各シナリオ感度

| シナリオ | MVPコスト | 影響 |
|---|---|---|
| 顧客0社（PoCのみ） | ¥100,000/月 | 想定通り |
| 顧客10社・想定通り | ¥125,000/月 | 想定通り |
| 顧客10社・トラフィック2x | ¥150,000/月 | +20% |
| 顧客10社・全員ロギング全文有効 | ¥180,000/月 | +44%（要価格反映） |

## 11. 結論

1. **MVPは月10〜15万円の AWS 費用**で運営可能
2. **損益分岐点は Standard 12社**、PRD初年度目標21社で粗利月30万円
3. **v1〜v2はインフラ比率25〜27%程度**で推移、規模拡大で単位コスト改善
4. **Shield Advanced は売上が見えるまで保留**（年3.6万USD ≒ 月60万円相当でMVPには重い）
5. **SOC2監査費用 200〜500万円は v1中盤に計上**、エンタープライズ顧客獲得への投資
